# v2.5 assist analysis is descriptive, without a league baseline

v2.5 computes assisted-make counts and bounded shares for the hero, including zone cross-sections, but does not build a league Case 3 baseline. A true comparison would require pulling and reconstructing every league game because the aggregate Case 2 creation payload cannot supply zone × assisted-make facts; that league-scale corpus is disproportionate before the descriptive feature proves its value.

Product copy may make absolute claims such as “most” or “almost all” when the worst-case assist bounds support them, but it may not imply “more than league,” “less self-created than average,” or any other absent comparison class. Existing Case 2 creation contexts retain their league baseline; league-relative Case 3 analysis is a separate future decision.
