# Assisted makes uses a bounded-share plot with a table twin

The Assisted Makes subsection renders a horizontal 0–100% bounded-share dot plot over the established zone hierarchy. Each dot is the assisted share among classified makes; when unknown makes exist, a line spans the minimum-to-maximum possible share over all makes. That interval encodes source coverage uncertainty only, never sampling confidence.

Every row carries a direct denominator, visual ink stays neutral rather than reusing the making palette, and no facts are hover-only. An accessible table is the complete numerical twin—makes, assisted, unassisted, unknown, coverage, classified share, and bounds—while the plot makes the zone pattern and the consequence of missing evidence immediately visible.

**Amendment (2026-07-16) - coverage diagnostics are conditional, not deleted.** When every displayed row is fully classified, `unknown` is zero, coverage is 100%, and each bound collapses to the classified assisted share. The product suppresses those three redundant table columns, their explanatory copy, and zero-width bound marks in that state. If any displayed row contains an unknown make, the full Unknown / Coverage / Bounds group and bounded-range explanation return together. The aggregation, report, verdict guards, and payload retain every coverage metric in both states. Attempt-empty refinement rows are also omitted from the product surface; a future nonempty band appears automatically.
